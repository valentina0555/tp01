¿que hacen, y como se utilizan las siguientes etiquetas?

>

La etiqueta <strong></strong> es para marcar las partes más importantes del texto

La etiqueta <i> </i> muestra el texto marcado en cursiva

La etiqueta <u> </u> muestra el texto subrayado


La etiqueta <br> </br> sirve para escribir un poema o una dirección, donde la división de las líneas es significante

La etiqueta <sub></sub> sirve para definir un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice

La etiqueta <sup></sup> es casi lo mismo que la etiqueta "sub", si no, que,el tramo principal del texto es en superíndice

La etiqueta <em></em> , sirve para marcar con énfasis las partes importantes de un texto. Puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis

La etiqueta <span></span> , sirve para aplicar estilo al texto o agrupar elementos en linea

La etiqueta <b></b> , sirve para indicar que el texto debe ser representado con una variable "bold", o "negrita", de la tipografia que se esté usando

a

La etiqueta <a></a> sirve para crear enlaces a otras páginas de internet, archivos o ubicaciones o cualquier otra URL
